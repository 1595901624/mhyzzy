# 麻花助手（Xposed Plugin）

麻花助手。当前是功能还不是很完善，支持**麻花影视
手机版2.5.0**版本，暂不支持 Exposed 框架，Virtual Xposed
未测试。

## 更新日志

* 麻花助手 0.1 Beta 2019/02/16
    1. 去除启动页广告；
    2. 去除启动页5s倒计时；
    3. 去除启动页历史广告显示；
    4. 去除“推荐”、“电影”、“电视剧”页面中存在的广告；
    5. 去除视频播放页面的广告。
    
## 如何安装
    
   软件运行需要 Xposed 环境，请安装 Xposed 框架后，在 Xposed
   中启用该框架。
